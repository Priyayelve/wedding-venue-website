💍 Wedding Venue Website

## 📌 Overview  
This project is a responsive multi-page Wedding Venue Website built using HTML, CSS, and JavaScript. It showcases a wedding venue’s services, events, and contact information with a clean and user-friendly interface.

---

## 🎯 Objective  
The goal of this project is to practice front-end development by building a structured and visually appealing website based on a real-world scenario.

---

## 🚀 Features  
- Home page with introduction  
- About page with venue details  
- Events page showcasing services  
- Contact page for inquiries  
- Responsive design for mobile and desktop  
- Clean navigation and layout  

---

## 🛠️ Technologies Used  
- HTML5  
- CSS3  
- JavaScript
